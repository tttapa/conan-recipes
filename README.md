# tttapa/conan-recipes

Recipes for the Conan package manager to build open-source packages for mathematical software and numerical optimization solvers.

- [blasfeo](https://github.com/giaf/blasfeo)
- [casadi](https://github.com/casadi/casadi)
- [coinmumps](https://github.com/coin-or-tools/ThirdParty-Mumps)
- [eigen](https://gitlab.com/libeigen/eigen)
- [guanaqo](https://github.com/tttapa/guanaqo)
- [hpipm](https://github.com/giaf/hpipm)
- [ipopt](https://github.com/coin-or/Ipopt)
- [ladel](https://github.com/kul-optec/LADEL)
- [qpalm](https://github.com/kul-optec/QPALM)
